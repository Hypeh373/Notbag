Get key data

Retrieve information about the bot key.

Request Body schema: application/json

key

required

string

Key of the bot

Responses

200 

Successful response

Get tasks for user

Retrieve tasks for the user. Resources are allocated for 1 hour.

Request Body schema: application/json

key

required

string

Key of the bot

user_id

required

integer

User ID

language_code

string

User's language code

limit

integer
